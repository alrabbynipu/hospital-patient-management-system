# Hospital Patient Management System

## Overview
A relational database system to manage hospital operations including 
patient records, doctor assignments, and admissions.
Designed and built using MySQL as an academic project.

## Schema Diagram
![Hospital Schema](hospital_schema_diagram.png)

## What's Included
- Database schema design with normalized tables
- Sample data insertion
- ETL process — data extraction, cleaning and transformation
- EDA — exploratory queries to analyze hospital data

## Tools Used
- MySQL
- MySQL Workbench
- draw.io (schema design)

## Files
- hospital_schema.sql — table creation and relationships
- hospital_data.sql — sample data
- hospital_analysis.sql — ETL and EDA queries

## How to Run
1. Open MySQL Workbench
2. Run hospital_schema.sql to set up the database
3. Run hospital_data.sql to load sample records
4. Run hospital_analysis.sql to explore the analysis
